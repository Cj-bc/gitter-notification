# gitter notifier for macOS

## dependencies

  - [mattn/gitter-cli](https://github.com/mattn/gitter-cli)
  -  [julienXX/terminal-notifier](https://github.com/julienXX/terminal-notifier)

# usage

1. wake up daemon

```
$ gitter-notifier start <room>
```

2. get notifications for each post at <room>

3. stop deamon

```
$ gitter-notifier stop
```
